# Introduction Notebooks

This directory contains Jupyter notebooks for the introductory college fair materials.

## Available Notebooks

### 1. college-fair-start.ipynb
**START HERE for college fair visitors**
- Welcome message and orientation
- Quick navigation to all intro topics
- Simple "Hello World" code example

### 2. what-is-cs.ipynb
**What is Computer Science?**
- Explains CS beyond just coding
- Shows real-world applications
- Career combinations (CS + other fields)
- No coding required - informational only

### 3. why-study-cs.ipynb
**Why Study Computer Science?**
- Career prospects and salaries
- Five great reasons to study CS
- CS Major vs Minor comparison
- No coding required - informational only

### 4. try-it-now.ipynb
**Try It Now! - Interactive Coding Demos**
- 5 hands-on coding demos
- Draw with code (turtle graphics)
- Chatbot interaction
- Random adventure generator
- Birthday calculator
- Password strength checker
- **🎯 Best for interactive booth demos!**

### 5. major-vs-minor.ipynb
**Major vs Minor Comparison**
- Detailed program comparison
- Course sequences
- Career paths for each option
- Sample combinations with other majors
- No coding required - informational only

## Usage

### For Students
1. **At College Fair**: Start with `college-fair-start.ipynb`
2. **Want to code immediately**: Go to `try-it-now.ipynb`
3. **Exploring career options**: Check `why-study-cs.ipynb`
4. **Choosing a program**: Read `major-vs-minor.ipynb`

### For Faculty/Staff
- Display `try-it-now.ipynb` on booth screens for live demos
- Print QR codes linking to these notebooks in JupyterLite
- Use `college-fair-start.ipynb` as the landing page

## Access Methods

All notebooks are available in three ways:

1. **JupyterLite** (browser-based, no installation):
   - `jupyterlite/content/[notebook-name].ipynb`
   
2. **Download** (for local Jupyter or Google Colab):
   - `files/intro/[notebook-name].ipynb`
   
3. **Website** (rendered as web pages):
   - `intro/[page-name].qmd` → rendered to HTML

## Note for Developers

These notebooks are designed to be:
- **Self-contained**: No dependencies beyond standard Python library
- **Informational-first**: Most focus on content, not complex code
- **College-fair friendly**: Appropriate for complete beginners
- **Quick**: Each takes 3-5 minutes to review

To update: Edit the corresponding `.qmd` file and regenerate, or edit the notebook directly.
